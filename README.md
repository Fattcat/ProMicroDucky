# ProMicroDucky
Supports SK lang keyboard
## Connection

### ProMicro - SD Module

- VCC  - VCC
- GND  - GND
- CS   - Pin 10
- MOSI - Pin 11
- MISO - Pin 12
- SCK  - Pin 13

### Button
- BUTTON_1: Pin 2 ─ 10kOhm - GND
- BUTTON_2: Pin 3 ─ 10kOhm - GND

### LED
- used LED_BUILTIN
